1. Utility Class = System.IO.Path  
Path เป็น Utility Class เนื่องจากเป็น static class ไม่สามารถสร้าง instance ได้ ใช้รวมกับ method สำหรับจัดการ path ของไฟล์ ไม่มีการเก็บ state ภายในคลาส

```csharp 
string path1 = @"c:\temp\MyTest.txt";
string path2 = @"c:\temp\MyTest";
string path3 = @"temp";

if (Path.HasExtension(path1))  //static method
{
    Console.WriteLine("{0} has an extension.", path1); 
}

```
https://learn.microsoft.com/en-us/dotnet/api/system.io.path?view=net-10.0

2. Mutable Class = System.Text.StringBuilder  
StringBuilder เป็น Mutable Class เพราะข้อมูลภายใน object สามารถเปลี่ยนแปลงได้ Method อย่างเช่น Append, Remove, Replace สามารถแก้ไขข้อมูลภายใน object เดิมได้
โดยใช้พื้นที่ใน buffer ที่ถูกจองไว้ล่วงหน้า (Capacity) ทำให้ Length เพิ่มขึ้นแต่ object ยังคงเดิม

```csharp 
StringBuilder sb = new StringBuilder("ABC", 50); //ค่าเริ่มต้นของ StringBuilder คือ "ABC" กำหนด capacity ไว้ 50 ตัวอักษร (จองไว้)
sb.Append(new char[] { 'D', 'E', 'F' }); // Append D, E, F ต่อท้าย string เดิม
```
https://learn.microsoft.com/en-us/dotnet/api/system.text.stringbuilder?view=net-10.0

3. Immutable Class = System.Net.Mail.MailAddress  
MailAddress เป็น Immutable Class เพราะเมื่อสร้าง object แล้วไม่สามารถแก้ไขค่าภายในได้ พวก Property ต่าง ๆ เช่น Address, DisplayName, User, Host เป็น read-only (ไม่มี setter) 
หากต้องการเปลี่ยนข้อมูล ต้อง สร้าง object ใหม่เท่านั้น

```csharp 
MailAddress from = new MailAddress("ben@contoso.com", "Ben Miller"); //ค่าอีเมลและชื่อถูกกำหนดผ่าน constructor
MailAddress to = new MailAddress("jane@contoso.com", "Jane Clayton"); //ถ้าต้องการเปลี่ยนค่า ต้องสร้าง MailAddress object ใหม่
```
https://learn.microsoft.com/en-us/dotnet/api/system.net.mail.mailaddress?view=net-10.0
